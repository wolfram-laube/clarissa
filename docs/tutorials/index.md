# Tutorials

Interactive Jupyter notebooks demonstrating CLARISSA capabilities.

## Available Tutorials

| Tutorial | Description | Topics |
|----------|-------------|--------|
| [Reservoir Basics](reservoir-basics.ipynb) | Introduction to reservoir simulation with CLARISSA | Darcy's Law, Pore Volume, Material Balance |

## Features

These notebooks include:

- **Mathematical Formulas** - LaTeX-rendered equations for reservoir engineering
- **Architecture Diagrams** - Mermaid flowcharts showing system design
- **Executable Code** - Python examples you can run and modify
- **ECLIPSE Examples** - Generated deck snippets

## Running Locally

To run notebooks locally with full interactivity:

```bash
pip install jupyter numpy
jupyter notebook docs/tutorials/
```

!!! note "Rendered vs Interactive"
    The notebooks on this site are statically rendered. For interactive execution, 
    clone the repository and run Jupyter locally.
