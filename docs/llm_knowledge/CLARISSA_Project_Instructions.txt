## Critical Technical Rules

- Mandatory section order: RUNSPEC → GRID → PROPS → SOLUTION → SUMMARY → SCHEDULE
- Bo MUST DECREASE with increasing pressure in PVDO (monotonicity)
- MNTH uses 3-letter codes (JAN, FEB, MAR) not numbers
- Grid sizing: 66 ft/cell for 10-acre quarter 5-spot pattern, NOT 200 ft/cell
- Use WELLSTRE stream name in WINJGAS, never inline compositions
- Use ELAPSED or specific DATES for time conditions, never DAY >= N
- WAG switching: Gas phase = WINJGAS + WCONINJE GAS; Water phase = WCONINJE WATER
- COMPDAT defaults notation: 2* means "skip 2 defaults", not "multiply by 2"

## After Every Simulation Run

Apply the 8-Point Validation Checklist:
1. OOIP matches hand calculation (within 5%)
2. Material balance: FOPT = FOIP(initial) − FOIP(final)
3. Pressure not negative, not increasing during production
4. Water cut increases (waterflood) or stays at Swc (depletion)
5. Production rates honor constraints
6. Injection ≈ Production (pressure maintenance)
7. No NaN or Inf values
8. Simulation time reasonable (not frozen at timestep 1)
